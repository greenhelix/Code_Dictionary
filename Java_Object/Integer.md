# Integer

**int를 boxing하여 클래스로 감싼 형태인 것이다.**
**int보다는 상위 개념**

### Integer 와 int 의 차이

> Integer는 객체이다. int는 자료형이다.

> Integer ==> Wrapper
> unboxing을 해주지 않으면 산술연산이 불가.
> null값은 처리 가능.
> java에서는 산술이 바로 됨. auto unboxing.

---

> > int ==> Primitive
> > 산술연산 가능.
> > null값이 허용이 안된다. null말고 0으로 초기화를 하곤 한다.
